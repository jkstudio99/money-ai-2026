### 🟡 MEDIUM (1-3 วัน)

- **ชื่อ:** `CondoBot` — LINE OA AI สำหรับนิติบุคคลคอนโด (MVP)
- **Architecture sketch:**
  ```
  LINE Webhook → FastAPI → Intent Classifier (Claude haiku)
                              ↓
              [ถามค่าน้ำ/ไฟ]  [แจ้งซ่อม]  [ถามข้อบังคับ]
                   ↓              ↓              ↓
              Supabase DB    Ticket System    RAG (PDF rules)
                   ↓              ↓              ↓
              LINE Reply ← ← ← format response ← Claude
  ```
- **Key components:**
  1. LINE Messaging API webhook handler
  2. Unit verification (ลูกบ้านต้อง verify เลขห้อง + LINE ID)
  3. RAG pipeline: load PDF ข้อบังคับนิติ → chunk → embed (OpenAI ada-002 หรือ Claude) → Supabase pgvector
  4. Maintenance ticket: form กรอก → store → dashboard สำหรับ staff นิติ
  5. Payment query: query Supabase table `unit_payments` → reply ค่าส่วนกลางค้างชำระ

- **ทักษะที่จะได้:**
  - LINE Messaging API (webhook, rich message, flex message)
  - RAG pipeline ตั้งแต่ต้นจนจบ
  - FastAPI + async Python
  - Supabase pgvector สำหรับ similarity search
  - Production deployment บน Railway
