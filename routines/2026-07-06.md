### 4. 🤖 AI Service / Micro-SaaS

- **Idea:** **ThaiDevCV** — AI Resume Builder เฉพาะสำหรับ Developer/Designer ไทยที่สมัครงานทั้งในและต่างประเทศ

  **Pain point ชัด:**
  - Dev ไทยส่วนใหญ่มี CV ที่เขียน "ผิด format" สำหรับ international job (LinkedIn, remote, startup SEA)
  - ไม่รู้ว่า keyword ATS (Applicant Tracking System) ควรใส่อะไร
  - แปล CV ไทย → English แล้วฟังดูแปลก (direct translation ไม่ใช่ professional English)
  - Canva template สวยแต่ ATS parse ไม่ได้ → ตกรอบโดยไม่มีคนอ่าน

- **Stack: build ใน weekend**
  - **Frontend:** Next.js 15 + shadcn/ui
  - **AI:** Claude claude-sonnet-5 API (structured output) → analyze experience → rewrite ด้วย STAR format + ATS keyword injection
  - **Template engine:** react-pdf สร้าง PDF ที่ ATS readable + human beautiful
  - **Auth + payment:** Supabase Auth + Stripe (หรือ PromptPay QR สำหรับตลาดไทย)
  - **Deploy:** Vercel (serverless functions ประหยัดค่า server)

- **Features MVP:**
  1. Paste LinkedIn profile หรือ text CV → AI rewrite ทันที
  2. เลือก target: Thai startup / International remote / Big tech / Freelance
  3. AI เติม quantified achievements: "พัฒนา feature X" → "Developed feature X reducing load time by 40% for 50k MAU"
  4. ATS score checker (regex-based keyword matching vs JD ที่ paste มา)
  5. Export PDF + JSON LinkedIn import format

- **Pricing + MRR target แรก:**
  - Free: 1 CV/เดือน (ล่อให้ทดลอง)
  - Pro: 149฿/เดือน (unlimited CV + ATS scan + 5 template)
  - Team: 499฿/เดือน (5 users + white-label สำหรับ coding bootcamp)
  - **MRR target แรก:** 50 Pro subscribers = 7,450฿/เดือน ใน 60 วัน
  - **Distribution:** Post ใน KBTG Tech, Agoda Engineering, Wongnai Dev community + LINE Group dev ไทย

## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1–2 ชม.)
- **ชื่อ:** `SplitAI` — AI-Powered Bill Splitter สำหรับกลุ่มเพื่อน
- **Problem:** หาร bill ร้านอาหารพร้อมกัน 6 คน — ใครกินอะไร ใครจ่ายเท่าไหร่ + tax + service charge คำนวณ manually = ทะเลาะกันทุกครั้ง
- **Stack:** React + Claude API (haiku — ถูก fast) + shadcn/ui
  - User paste receipt text (หรือ upload รูป → Tesseract.js OCR)
  - Claude parse items → assign ให้แต่ละคน → คำนวณ final amount รวม VAT 7% + service charge
  - Output: สรุป "คุณต้องโอน X บาทให้ [คนจ่ายก่อน]" + QR PromptPay link
- **ทำไมขายได้:** ปัญหา universal ทุกวัน; monetize ด้วย AdSense หรือ freemium (unlimited history = 29฿/เดือน); viral เพราะ share result ให้เพื่อน
