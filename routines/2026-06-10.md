### 🟡 MEDIUM (1-3 วัน)
- **ชื่อ:** DifyLine Starter — Open-source LINE OA Chatbot template
- **Architecture sketch:**
  ```
  LINE User Message
       ↓
  LINE Messaging API (webhook)
       ↓
  Next.js API Route (/api/line-webhook)
       ↓
  Dify Workflow API (intent → knowledge base → response)
       ↓
  Reply to LINE + log booking to Supabase
       ↓
  Google Calendar API (create event for appointment)
  ```
- **Key components:**
  - Dify workflow: intent classification (จอง / ถามราคา / ยกเลิก / ทั่วไป) → branch routing
  - Knowledge Base: upload PDF menu หรือ service list → Dify auto-chunk + embed
  - Booking DB: Supabase table `bookings` (customer_name, service, datetime, status)
  - Calendar sync: Google Calendar API ฟรี → สร้าง event อัตโนมัติ
  - Admin view: Notion integration ดู bookings ของวันนี้ใน Notion database
- **ทักษะที่จะได้:** Dify workflow design, LINE Messaging API webhook, Supabase CRUD + real-time, prompt engineering ภาษาไทย, Google Calendar API, mini SaaS deployment บน Vercel
