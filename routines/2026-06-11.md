- **ทำไมขายได้:** deploy เป็น Telegram Bot → charge ฿19/เดือน; หรือทำ free viral tool → traffic → affiliate sport gear ใน bio; สาย sports enthusiast ไทยชอบมาก algorithm ทดสอบจริงผ่าน match วันนี้

```javascript
// predict.js — รัน: node predict.js "Germany" "Scotland"
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic();
const [,, teamA, teamB] = process.argv;

const response = await client.messages.create({
  model: "claude-haiku-4-5-20251001",
  max_tokens: 512,
  messages: [{
    role: "user",
    content: `วิเคราะห์การแข่งขัน FIFA World Cup 2026: ${teamA} vs ${teamB}
    พิจารณา: ฟอร์มล่าสุด, head-to-head, ความแข็งแกร่งของนักเตะหลัก
    ตอบเป็นภาษาไทย: ทีมที่คาดว่าจะชนะ + เหตุผล 3 ข้อ + สกอร์ที่คาด (ระบุว่าเป็นการวิเคราะห์ ไม่ใช่ผลจริง)`
  }]
});

console.log(response.content[0].text);
```
