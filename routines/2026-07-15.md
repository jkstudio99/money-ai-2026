## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **InstaAffiliateGen** — Instagram × Shopee Affiliate Caption Generator
- **Problem:** Creator ไทยที่เพิ่งเข้า Instagram × Shopee Affiliate ต้องเขียน caption ใหม่ทุก post — ใช้เวลานาน ไม่รู้ว่าต้องใส่ hashtag อะไร, CTA แบบไหนดี
- **Stack:** Python 3.12 + Claude API (`claude-sonnet-5`) + Streamlit — UI input: ชื่อสินค้า, ราคา, target audience (นักเรียน/พนักงาน/แม่บ้าน), affiliate URL → output: caption ภาษาไทย 3 version (funny/informative/lifestyle) + hashtag set 20 อัน
- **ทำไมขายได้:** ปิดตลาด early adopter Instagram × Shopee ที่กำลังหา workflow — โปรโมทฟรีผ่าน TikTok "AI เขียน caption Shopee affiliate ให้ในคลิปเดียว" แล้วขาย Gumroad 99 บาท หรือให้ฟรีแล้ว monetize ด้วย affiliate ตัวเอง

```python
# sample core function
import anthropic

def generate_captions(product_name: str, price: int, audience: str, affiliate_url: str) -> dict:
    client = anthropic.Anthropic()
    prompt = f"""
    สร้าง Instagram caption ภาษาไทย 3 แบบสำหรับ Shopee affiliate post:
    - สินค้า: {product_name} ราคา {price} บาท
    - กลุ่มเป้าหมาย: {audience}
    - Affiliate URL: {affiliate_url}
    
    แบบที่ 1 (funny/meme tone) | แบบที่ 2 (informative) | แบบที่ 3 (lifestyle)
    แต่ละแบบ: body 3-4 บรรทัด + emoji + CTA + 20 hashtags
    """
    message = client.messages.create(
        model="claude-sonnet-5",
        max_tokens=1500,
        messages=[{"role": "user", "content": prompt}]
    )
    return {"captions": message.content[0].text}
```
