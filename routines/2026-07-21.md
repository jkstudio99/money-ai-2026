## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **`KpopCardCLI`** — AI Photocard Caption & Affiliate Link Generator
- **Problem:** แฟนคลับต้องพิมพ์ caption เอง + หา affiliate link เอง ทำให้ slow ลง → bot จัดการให้
- **Stack:** Python 3.12 + `anthropic` SDK (Claude Haiku) + Shopee Affiliate API (REST) + `rich` library (pretty CLI output) + `.env` config
- **ทำไมขายได้:** ขาย CLI script ฿299 ครั้งเดียวผ่าน Gumroad — target: แฟนคลับที่ขาย K-pop merch ออนไลน์ (กลุ่ม Facebook "K-pop Thailand" 3M+ member)
- **Code Sketch:**
  ```python
  # รับ product URL จาก Shopee → ดึง product info → Claude gen caption → gen affiliate link → copy to clipboard
  import anthropic, requests, pyperclip
  
  def gen_caption(product_name: str, idol: str) -> str:
      client = anthropic.Anthropic()
      msg = client.messages.create(
          model="claude-haiku-4-5-20251001",
          max_tokens=300,
          messages=[{"role": "user", "content": f"เขียน TikTok caption ภาษาไทย FOMO สำหรับ {product_name} ของ {idol} ใส่ emoji + 10 hashtag"}]
      )
      return msg.content[0].text
  ```
