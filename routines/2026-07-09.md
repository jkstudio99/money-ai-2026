## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)
- **ชื่อ:** RainAffBot — ส่ง Shopee affiliate link อัตโนมัติเมื่อฝนกำลังจะตก
- **Problem:** คนไทยไม่ได้ซื้อร่ม/เสื้อกันฝน/เครื่องฟอกอากาศล่วงหน้า — ซื้อเมื่อเดือดร้อน → moment-based affiliate notification = conversion rate สูงกว่า 3–5×
- **Stack:** Python 3.12 + OpenWeatherMap API (free tier) + LINE Notify API + schedule (cron job) + `.env` สำหรับ API keys
- **Logic:** ดึง weather forecast ทุก 3 ชม. สำหรับ Bangkok (lat/lon) → ถ้า rain probability > 60% ในช่วง 3 ชม. ข้างหน้า → ส่ง LINE Notify พร้อมข้อความ + affiliate link ร่มกันน้ำ Shopee → log ทุก send ป้องกัน spam
- **ทำไมขายได้:** intent-based timing — คนรับ push แล้วเปิดซื้อเลย; ต้นทุน 0฿/เดือน (free tier ทั้งหมด); scale ได้เป็น SaaS affiliate alert service ในภายหลัง

```python
# ตัวอย่าง core logic
import requests, os
from dotenv import load_dotenv

load_dotenv()
OWM_KEY = os.getenv("OWM_API_KEY")
LINE_TOKEN = os.getenv("LINE_NOTIFY_TOKEN")
SHOPEE_LINK = os.getenv("SHOPEE_AFFILIATE_URL")  # ลิงก์ร่มกันฝน

def check_rain_and_notify():
    r = requests.get(
        "https://api.openweathermap.org/data/2.5/forecast",
        params={"lat": 13.75, "lon": 100.52, "appid": OWM_KEY, "cnt": 3, "units": "metric"}
    )
    forecasts = r.json()["list"]
    max_rain_prob = max(f.get("pop", 0) for f in forecasts)
    if max_rain_prob > 0.6:
        msg = f"\n🌧️ ฝนกำลังจะตก! ({int(max_rain_prob*100)}% probability)\nซื้อร่มกันฝนก่อนออกจากบ้าน → {SHOPEE_LINK}"
        requests.post("https://notify-api.line.me/api/notify",
            headers={"Authorization": f"Bearer {LINE_TOKEN}"},
            data={"message": msg})
```
