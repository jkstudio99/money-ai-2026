- **ชื่อ:** `LineOrderBot` — LINE Chatbot รับออเดอร์ + ออก invoice อัตโนมัติ
- **Architecture sketch:**
  ```
  LINE OA → Webhook → Express.js server (Vercel)
      → Claude haiku (parse intent: สั่งสินค้า / เช็คสถานะ / ถามราคา)
      → Supabase (products table, orders table)
      → Response LINE + generate PDF invoice (PDFKit)

  ```
- **Key components:** LINE Messaging API, Claude tool use สำหรับ parse คำสั่งไทย, Supabase CRUD, PDFKit สร้างใบเสร็จ
