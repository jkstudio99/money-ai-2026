- **Stack:** Python 3.12, `anthropic` SDK (claude-sonnet-5 ราคา $2/1M input tokens), `httpx` ดึง fixture จาก football-data.org API (ฟรี tier), `rich` library สำหรับ terminal output สวย
- **Code ที่ต้องเขียน:**
  ```python
  import anthropic, httpx
  from rich.console import Console

  def get_today_matches():
      r = httpx.get("https://api.football-data.org/v4/competitions/WC/matches",
                    params={"status": "SCHEDULED", "dateFrom": TODAY, "dateTo": TODAY},
                    headers={"X-Auth-Token": API_KEY})
      return r.json()["matches"]

  def predict_match(home, away):
      client = anthropic.Anthropic()
      msg = client.messages.create(
          model="claude-sonnet-5",
          max_tokens=512,
          messages=[{"role": "user", "content":
              f"วิเคราะห์ {home} vs {away} FIFA 2026 knockout: form, head-to-head, key players — "
              f"predict: winner, score, confidence%. ภาษาไทย casual 5 bullet points"}]
      )
      return msg.content[0].text
  ```
