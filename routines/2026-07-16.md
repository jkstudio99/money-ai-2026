## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `FitPlan CLI` — AI Workout + Meal Plan Generator แบบ command-line
- **Problem:** ต้องการ personalized plan แต่ app fitness ทั่วไปไม่ยืดหยุ่น; ต้องจ่ายแพง
- **Stack:** Python 3.12 + `anthropic` SDK + `rich` (pretty terminal output) + `typer` (CLI args)
- **วิธีทำ:**
```python
import anthropic, typer
from rich.markdown import Markdown
from rich.console import Console

app = typer.Typer()
console = Console()

@app.command()
def generate(
    goal: str = typer.Option(..., prompt="เป้าหมาย (ลดไขมัน/เพิ่มกล้าม/endurance)"),
    days: int = typer.Option(7, help="จำนวนวัน"),
    equipment: str = typer.Option("ไม่มีอุปกรณ์", prompt="อุปกรณ์ที่มี"),
    level: str = typer.Option("beginner", prompt="ระดับ (beginner/intermediate/advanced)")
):
    client = anthropic.Anthropic()
    prompt = f"""สร้าง {days}-day workout + meal plan สำหรับ:
- เป้าหมาย: {goal}
- อุปกรณ์: {equipment}
- ระดับ: {level}
Format: Markdown table แยก Day/Workout/Duration/Meal Plan/Calories
ภาษาไทย, practical, ทำได้จริง"""

    message = client.messages.create(
        model="claude-sonnet-5",
        max_tokens=2048,
        messages=[{"role": "user", "content": prompt}]
    )
    console.print(Markdown(message.content[0].text))

if __name__ == "__main__":
    app()
```
- **ทำไมขายได้:** share บน GitHub → TikTok demo → ดึง traffic → upsell ThaiCV AI หรือ LINE OA bot; portfolio piece ที่ใช้ Claude API จริง
