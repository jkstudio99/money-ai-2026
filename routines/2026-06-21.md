### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `campkit-cli` — AI Camping Packing List Generator
- **Problem:** ต้องค้นหา packing list camping จากหลายแหล่ง ใช้เวลา 30+ นาที และได้ list ทั่วไปไม่ personalized
- **Stack:** Python 3.12 + `anthropic` SDK + `rich` (terminal formatting) + `typer` CLI framework
- **How it works:** พิมพ์ `python campkit.py generate --days 3 --people 2 --season rainy --level beginner` → Claude generate packing list ส่วนตัว → output เป็น markdown table สวยงามใน terminal + บันทึกเป็น `trip-plan.md`

```python
# core prompt
SYSTEM = """คุณเป็น outdoor expert ไทยที่รู้จักสภาพอากาศและ trail ไทยดี
สร้าง packing list camping ที่ practical ไม่เกินจำเป็น เรียงตามลำดับความสำคัญ"""

USER = f"""Trip: {days} วัน, {people} คน, ฤดู: {season}, ระดับ: {level}
สร้าง packing list แบบ markdown table: หมวด | รายการ | จำนวน | หมายเหตุสำคัญ"""
```

- **ทำไมขายได้:** โพสต์ใน Facebook Group "Camping Thailand" (300k+) + r/Thailand → GitHub traffic → promote TrailAI SaaS ผ่าน README
