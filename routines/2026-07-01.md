## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `CoffeeAffiliateBot` — AI Coffee Review & Affiliate Content Generator

- **Problem:** สร้างบทรีวิวกาแฟ + copy สำหรับ affiliate post แบบ batch ต้องใช้เวลานานถ้าทำมือ; developer ที่มี Shopee affiliate account แต่ไม่ถนัดเขียน copy ภาษาไทย

- **Stack:** Python 3.12 + `anthropic` SDK + `rich` (pretty output) + dotenv

- **Core logic (pseudocode):**
  ```python
  products = [
    {"name": "Bialetti Moka Express 3-cup", "price": 390, "url": "..."},
    {"name": "Timemore C3 Pro Grinder", "price": 2200, "url": "..."},
  ]
  for p in products:
      review = claude.messages.create(
          model="claude-sonnet-5",
          messages=[{"role":"user","content":
              f"เขียนรีวิวกาแฟ {p['name']} ราคา {p['price']}฿ ภาษาไทย informal 5 บรรทัด "
              f"เน้น: ใช้ง่าย, คุ้มค่า, เหมาะคนเริ่มต้น จบด้วย CTA 'ซื้อได้ที่ Shopee {p['url']}'"
          }]
      )
      print(review.content[0].text)
  ```

- **ทำไมขายได้:** ตลาด specialty coffee ไทย boom ตาม trend; ไม่ต้องมีร้านค้า แค่ affiliate link + content; สามารถ scale เป็น Facebook Page หรือ LINE OA กาแฟได้ทันที
