## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1–2 ชม.)

- **ชื่อ:** `FlashGen` — AI Flashcard Generator จาก URL/Text
- **Problem:** นักเรียน/นักศึกษาเสียเวลาทำ flashcard เอง จาก lecture notes หรือ Wikipedia
- **Stack:**
  - Python script + Claude API (Haiku 4.5 — ถูกที่สุด)
  - Input: URL หรือ paste text → chunk → generate Q&A JSON
  - Output: export เป็น CSV import เข้า Anki ได้เลย
  ```python
  # core loop
  import anthropic, httpx
  client = anthropic.Anthropic()

  def generate_flashcards(text: str, n: int = 10) -> list[dict]:
      response = client.messages.create(
          model="claude-haiku-4-5-20251001",
          max_tokens=1024,
          messages=[{
              "role": "user",
              "content": f"สร้าง {n} flashcard จากข้อความนี้ ในรูปแบบ JSON array [{{\"q\":\"...\",\"a\":\"...\"}}]\n\n{text[:3000]}"
          }]
      )
      return response.content[0].text
  ```
- **ทำไมขายได้:** ต่อยอดเป็น StudyDeck AI (PART 1 #4) ได้ทันที; ใช้เป็น demo prototype
