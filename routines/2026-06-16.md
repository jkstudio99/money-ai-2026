## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.) — "RainMood" ฝนตก + Spotify Playlist Generator

- **ชื่อ:** `RainMood` — เปิดแอปวันฝนตก → AI เลือก playlist ให้ตาม mood + weather
- **Problem:** คนไทยชอบฟังเพลงตอนฝนตก แต่ไม่รู้จะเลือก playlist อะไร — ต้องสลับ Spotify ทุกครั้ง
- **Stack:** Python 3.12 + `requests` (OpenWeatherMap API ฟรี) + Spotify Web API (OAuth) + Claude API (mood classification)
- **Core logic:**
```python
# 1. Get weather for Bangkok
weather = get_weather("Bangkok")  # rain, cloudy, sunny

# 2. Ask Claude to pick mood
mood = claude.classify(f"Weather: {weather}. Pick playlist mood: chill/sad/upbeat/focus")

# 3. Search Spotify playlist by mood
playlist = spotify.search(f"Thai {mood} rainy day 2026", type="playlist", limit=1)

# 4. Open in browser
webbrowser.open(playlist['external_urls']['spotify'])
```
- **ทำไมขายได้:** ต่อยอดเป็น LINE bot "RainMoodBot" → ส่ง playlist อัตโนมัติทุกเช้าวันฝนตก → charge 49฿/เดือน; หรือ affiliate Spotify Premium link
