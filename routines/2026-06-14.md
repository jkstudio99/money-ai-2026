## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `TuyaTimer-CLI` — Smart Plug World Cup Scheduler
- **Problem:** ดูบอลโลก 2026 ยาวถึงตี 2 แล้วลืมปิดไฟ/เครื่องใช้ไฟฟ้า → ค่าไฟพุ่ง + เสี่ยงไฟไหม้
- **Stack:** Python 3 + `tinytuya` (Tuya cloud API wrapper) + `schedule` + `python-dotenv` + JSON config
- **ทำไมขายได้:** build เป็น tutorial video "ควบคุม smart plug ด้วย Python ใน 60 นาที" → embed Shopee affiliate link ไป smart plug SKU ที่ใช้ใน tutorial → ทุกคนที่ดู video แล้วซื้อ plug = passive commission; GitHub repo เพิ่ม credibility portfolio

```python
# main.py skeleton
import tinytuya, schedule, time
from dotenv import load_dotenv

load_dotenv()

def turn_off_plug(device_id: str):
    d = tinytuya.OutletDevice(device_id, ...)
    d.turn_off()
    print(f"[TuyaTimer] {device_id} ปิดแล้ว")

# ปิดทุกอุปกรณ์ตี 2 (หลังบอลจบ)
schedule.every().day.at("02:00").do(turn_off_plug, device_id="YOUR_DEVICE_ID")

while True:
    schedule.run_pending()
    time.sleep(30)
```
