- **Feature หลัก:**
  ```python
  # input: list of Shopee product URLs
  # output: markdown file with affiliate links + AI descriptions
  products = ["url1", "url2", "url3"]
  for url in products:
      product_info = scrape_product(url)
      description = claude_haiku.generate_thai_desc(product_info, season="rainy")
      affiliate_link = shopee_api.get_affiliate_link(url)
      markdown_output.append(f"**{product_info.name}** — {description}\n[ซื้อเลย]({affiliate_link})")
  ```
