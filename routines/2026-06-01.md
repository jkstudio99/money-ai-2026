## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **VibePomodoro** — Pomodoro Timer พร้อม AI Task Breakdown
- **Problem:** นักพัฒนาไทยมักเริ่มงานไม่ได้เพราะ task ใหญ่เกินไป → Claude ช่วย break task ให้เป็น 25-min chunks อัตโนมัติ
- **Stack:** React + Vite + Claude API (`claude-haiku-4-5` สำหรับ cost ต่ำ) + localStorage; deploy บน Vercel ฟรี
- **ทำไมขายได้:** ของฟรีใช้ดีจริง → บอกปากต่อปาก dev community → monetize ด้วย "Pro" tier ที่ export task log เป็น PDF สำหรับ freelancer billing (99฿/เดือน)

```
// Core prompt pattern:
const breakdownPrompt = `แบ่งงานนี้: "${task}" 
ออกเป็น Pomodoro sessions (25 นาที/session) 
ให้ deliverable ชัดในแต่ละ session 
ตอบเป็น JSON array [{session: 1, goal: "..."}]`
```
