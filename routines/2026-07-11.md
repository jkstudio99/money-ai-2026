## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `PetPriceWatch` — CLI tool ติดตามราคา pet product บน Shopee
- **Problem:** pet owner ต้องการรู้ว่าอาหารสัตว์/อุปกรณ์ตัวใดลดราคาและเมื่อไหร่
- **Stack:** Python 3.12 + `httpx` + `rich` (pretty table) + SQLite (เก็บ price history) + cron job daily
- **Core logic:**
  ```python
  # shopee public API (unofficial, no auth needed for public listings)
  import httpx, sqlite3
  from rich.table import Table
  from rich.console import Console

  def fetch_price(keyword: str, limit: int = 5):
      url = "https://shopee.co.th/api/v4/search/search_items"
      params = {"keyword": keyword, "limit": limit, "order": "desc", "page_type": "search"}
      r = httpx.get(url, params=params, headers={"User-Agent": "Mozilla/5.0"})
      items = r.json().get("items", [])
      return [(i["item_basic"]["name"], i["item_basic"]["price"] / 100000) for i in items]

  def check_and_alert(watchlist: list[str]):
      console = Console()
      table = Table(title="Pet Price Watch 🐾")
      table.add_column("Product", style="cyan")
      table.add_column("Price (฿)", style="green")
      for kw in watchlist:
          for name, price in fetch_price(kw):
              table.add_row(name[:50], f"{price:,.0f}")
      console.print(table)

  if __name__ == "__main__":
      check_and_alert(["GPS pet tracker", "airline pet carrier", "grain free cat food"])
  ```
- **ทำไมขายได้:** ต่อยอดเป็น Telegram bot แจ้งเตือนราคา → monetize ด้วย Shopee affiliate link embed
