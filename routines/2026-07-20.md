## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **`SmartHomeDeal CLI`** — Shopee Smart Home Affiliate Tracker แบบ command-line
- **Problem:** affiliate marketer ต้องเปิด Shopee ทุกวัน เช็ค price drop + commission เปลี่ยน → เสียเวลา ทำไม่สม่ำเสมอ
- **Stack:** Python 3.12 + `httpx` (async HTTP) + `rich` library (pretty table) + `typer` (CLI) + dotenv (Shopee Affiliate API key) + `schedule` (run ทุกเช้า 8:00)
- **Feature:**
  ```
  smarthome-deal check --category smart-home --min-commission 5
  smarthome-deal track add --url "https://shopee.co.th/..."
  smarthome-deal report --output affiliate_report.csv
  ```
- **Output:** table แสดง product name | price | commission % | 7-day price history | affiliate link — auto copy link ลง clipboard
- **ทำไมขายได้:** affiliate marketer 10,000+ คนในไทยต้องการ tool นี้; ปล่อยเป็น open source บน GitHub → traffic จาก Google; ต่อยอดเป็น SaaS ฿99/เดือน
