## 🎯 ภารกิจ 60 นาทีวันนี้

> **เลือก: RainCheck TH (🟢 EASY)** — สร้าง Python script เชื่อม Open-Meteo API + Claude API + LINE Notify ส่ง morning weather alert พร้อม 2 affiliate link rain gear วันนี้เลย

**Deliverable ชัดเจน:**
1. `rain_check.py` รันได้ → ส่ง LINE message "วันนี้ฝนตก X% โอกาส — ซื้อก่อนเลย: [Shopee link]"
2. ตั้ง cron job (หรือ GitHub Actions schedule) รันตี 6:30 ทุกวัน
3. ถ้าเหลือเวลา: ต่อ Shopee Affiliate link จริง 2 ชิ้น (ร่มกันฝน + rain repellent) → track click ด้วย UTM parameter

**เริ่มด้วย:**
```python
import httpx, os
from anthropic import Anthropic

client = Anthropic()
# Open-Meteo Bangkok: lat=13.75, lon=100.52
resp = httpx.get("https://api.open-meteo.com/v1/forecast?latitude=13.75&longitude=100.52&daily=precipitation_probability_max&timezone=Asia%2FBangkok&forecast_days=1")
rain_pct = resp.json()["daily"]["precipitation_probability_max"][0]

msg = client.messages.create(
    model="claude-opus-4-8",
    max_tokens=200,
    messages=[{"role": "user", "content": f"ฝนวันนี้กทม โอกาส {rain_pct}% เขียน LINE message สั้น 2 บรรทัด แนะนำซื้อของกันฝน tone เป็นกันเอง ใส่ emoji 1-2 ตัว"}]
)
print(msg.content[0].text)
```
