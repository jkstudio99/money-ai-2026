### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** PromptPay Slip Analyzer
- **Problem:** รับภาพ slip โอนเงิน PromptPay → Claude Vision API extract ข้อมูล (จำนวนเงิน, ชื่อผู้รับ, เวลา, เลขอ้างอิง, ธนาคาร) → output JSON/CSV → ช่วย freelancer และ SME ไทยสรุปบัญชีรับเงินอัตโนมัติ
- **Stack:** Python 3.12 + Anthropic SDK (claude-sonnet-4-6 vision) + PIL/base64 image encoding + JSON output + argparse CLI
- **ทำไมขายได้:** freelancer/SME ไทยต้องการ automation สรุป slip รายวัน — ปัจจุบันทำมือหรือนับผิด; ต่อยอดเป็น Telegram bot subscription ฿49/เดือน หรือ lifetime access ฿299; ใช้เวลาสร้าง 1–2 ชั่วโมง แต่ขายได้ทันที

```python
# ตัวอย่าง core logic
import anthropic, base64, json, sys

def analyze_slip(image_path: str) -> dict:
    client = anthropic.Anthropic()
    with open(image_path, "rb") as f:
        img_data = base64.standard_b64encode(f.read()).decode()
    
    response = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=512,
        messages=[{
            "role": "user",
            "content": [
                {"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": img_data}},
                {"type": "text", "text": "Extract from this PromptPay slip: amount (number), recipient_name, sender_bank, datetime (ISO8601), reference_number. Return JSON only, no markdown."}
            ]
        }]
    )
    return json.loads(response.content[0].text)

if __name__ == "__main__":
    result = analyze_slip(sys.argv[1])
    print(json.dumps(result, ensure_ascii=False, indent=2))
```
