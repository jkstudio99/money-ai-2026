## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)
- **ชื่อ:** `LINE Order Collector` — บอทรับออเดอร์ LINE แล้วยัดเข้า Google Sheet อัตโนมัติ
- **Problem:** เจ้าของร้านรับออเดอร์ทาง LINE chat แล้วต้องพิมพ์ลง Excel เอง — เสียเวลา 30–60 นาที/วัน
- **Stack:** Python 3.12 + LINE Messaging API (Webhook) + Google Sheets API (gspread) + Claude API (parse ออเดอร์ free-text → structured JSON); deploy บน Railway free tier
- **MVP Flow:**
  ```
  ลูกค้า LINE message → Webhook → Claude parse:
  {"item":"resistance band","qty":2,"size":"M","address":"..."} 
  → append Google Sheet row → reply confirm อัตโนมัติ
  ```
- **ทำไมขายได้:** ร้านค้า LINE ไทยมีหลายแสนร้าน, pain point ชัด, ขายเป็น template + tutorial 299฿ หรือ offer setup service 1,500฿/ร้าน
