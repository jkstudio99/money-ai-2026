## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `FIFAPredictor.th` — Claude-Powered World Cup Group Stage Predictor
- **Problem:** คนอยากรู้ว่า AI คิดว่าใครจะผ่าน group stage — แต่ไม่มี tool ภาษาไทยที่ให้ reasoning ชัดเจน
- **Stack:** Python + Claude API (`claude-opus-4-8`) + rich terminal output หรือ Streamlit (1 file, deploy ฟรีบน Streamlit Community Cloud)
- **Core logic:**
  ```python
  import anthropic

  client = anthropic.Anthropic()

  def predict_group(group: str, teams: list[str]) -> str:
      prompt = f"""วิเคราะห์ FIFA World Cup 2026 Group {group}
  ทีม: {', '.join(teams)}
  ข้อมูลพิจารณา: FIFA ranking, recent form, head-to-head, injury news
  Output: ทีมที่จะผ่าน 2 อันดับแรก + เหตุผล 3 ข้อต่อทีม (ภาษาไทย)"""

      message = client.messages.create(
          model="claude-opus-4-8",
          max_tokens=1024,
          messages=[{"role": "user", "content": prompt}]
      )
      return message.content[0].text

  # ตัวอย่าง Group A
  result = predict_group("A", ["USA", "Mexico", "Canada", "Uruguay"])
  print(result)
  ```
- **ทำไมขายได้:** content TikTok ทำได้ทันที + shareable ใน Facebook group บอล + FIFA hype อีก 7 วัน
