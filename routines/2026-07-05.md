## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **SkincareCopyAI** — Thai TikTok Caption Batch Generator สำหรับ Skincare Seller

- **Problem:** Skincare seller ไทยต้องเขียน caption TikTok ใหม่ทุกวัน — เสียเวลา 30–60 นาที/วัน เขียน 5–10 posts แตกต่างกัน แต่ message เดียวกัน

- **Stack:**
  ```python
  # สคริปต์ Python 50 บรรทัด
  import anthropic, json, sys

  client = anthropic.Anthropic()

  def generate_captions(product_name: str, benefit: str, n: int = 5) -> list[str]:
      prompt = f"""
      สร้าง {n} TikTok caption ภาษาไทย สำหรับ skincare product:
      Product: {product_name}
      Benefit: {benefit}
      Rules: แต่ละ caption ต้องมี hook แตกต่างกัน, 60–80 คำ,
      ใส่ 5 hashtag relevant ท้าย, CTA ที่ต่างกัน (DM/ลิงก์ bio/comment)
      Output: JSON array of strings
      """
      msg = client.messages.create(
          model="claude-sonnet-5",
          max_tokens=2000,
          messages=[{"role": "user", "content": prompt}]
      )
      return json.loads(msg.content[0].text)

  if __name__ == "__main__":
      product = sys.argv[1] if len(sys.argv) > 1 else "Niacinamide Serum"
      benefit = sys.argv[2] if len(sys.argv) > 2 else "ลดรูขุมขน ผิวใส"
      captions = generate_captions(product, benefit)
      for i, c in enumerate(captions, 1):
          print(f"\n--- Caption {i} ---\n{c}")
  ```

- **ทำไมขายได้:** Skincare seller TikTok ไทยมีหลายพัน คน; ขาย script บน Gumroad 99–199฿ หรือต่อยอดเป็น web app subscription 149฿/เดือน; cost to build < 2 ชม.
