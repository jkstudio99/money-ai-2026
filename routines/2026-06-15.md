## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.) — "ThaiQuote" ใบเสนอราคา PDF ภาษาไทยใน 1 คลิก

- **ชื่อ:** ThaiQuote CLI / Web form
- **Problem:** freelancer ไทยเสียเวลา 30 นาที ต่อ quote → ลูกค้ารอนาน → deal หาย
- **Stack:** Node.js + `pdfkit` หรือ Python + `reportlab`; input: ชื่อลูกค้า, บริการ, ราคา/ชั่วโมง, จำนวนชั่วโมง, due date → output: PDF invoice ภาษาไทย พร้อม PromptPay QR code
- **ขั้นตอน:**
  ```bash
  npm init && npm install pdfkit qrcode
  # generate-invoice.js: รับ JSON input → render Thai font (Sarabun) + PromptPay QR
  node generate-invoice.js --client "บริษัท ทดสอบ จำกัด" --amount 15000 --service "Web Development"
  ```
- **ทำไมขายได้:** freelancer ไทย > 500,000 คน, ทุกคนต้องการ — ขายเป็น Chrome Extension / web app 99฿/ครั้ง หรือ 199฿/เดือน unlimited

- **Architecture sketch:**
  ```
  ลูกค้า DM LINE OA
       ↓
  LINE Messaging API webhook
       ↓
  Express.js server (Vercel/Railway)
       ↓
  Claude API (intent detection: ถามราคา/ถาม portfolio/นัด call)
       ↓
  Response: ส่ง portfolio link / ราคากลาง / Calendly link
       ↓
  ถ้า intent = "ต้องการจ้าง" → notify freelancer ทาง LINE push
  ```
- **Key components:**
  - LINE Messaging API (ฟรี tier 500 msg/month)
