### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** `WorldCupAlertBot` — FIFA 2026 Match Score Alert ส่ง LINE Notify
- **Problem:** developer ไทยที่กำลัง code อยู่ไม่ได้เปิด TV/แอปตลอดเวลา อยากรู้ผลบอลทันทีที่จบครึ่งหรือจบเกม
- **Stack:** Python 3.12 + `requests` + football-data.org API (free tier 10 req/min) + LINE Notify API (ฟรี)
- **ทำไมขายได้:** push ขึ้น GitHub public → share บน dev community ไทย → viral ช่วง World Cup → ต่อยอดเป็น affiliate ของ merchandise football

```python
# match_alert.py — blueprint
import requests, os
FOOTBALL_API_KEY = os.environ["FOOTBALL_API_KEY"]
LINE_TOKEN = os.environ["LINE_TOKEN"]
MATCH_ID = "2026-07-01-ROUND16"  # ตัวอย่าง

def get_score(match_id):
    r = requests.get(
        f"https://api.football-data.org/v4/matches/{match_id}",
        headers={"X-Auth-Token": FOOTBALL_API_KEY}
    )
    return r.json()

def send_line(msg):
    requests.post(
        "https://notify-api.line.me/api/notify",
        headers={"Authorization": f"Bearer {LINE_TOKEN}"},
        data={"message": msg}
    )

match = get_score(MATCH_ID)
score = f"{match['homeTeam']['name']} {match['score']['fullTime']['home']}–{match['score']['fullTime']['away']} {match['awayTeam']['name']}"
send_line(f"\n⚽ ผลบอล FIFA 2026\n{score}\nสถานะ: {match['status']}")
```
