## 🛠️ PART 2: Daily Coding Project (3 ระดับ)

### 🟢 EASY (1-2 ชม.)

- **ชื่อ:** **PetProductBot** — Shopee Pet Affiliate Content Generator
- **Problem:** สร้าง review content สำหรับ pet products ด้วยมือใช้เวลา 2-3 ชั่วโมง/โพสต์ — scale ไม่ได้
- **Stack:** Python 30 บรรทัด + Claude API (`claude-sonnet-5`)
  ```python
  import anthropic, json

  client = anthropic.Anthropic()

  def generate_pet_review(product_name: str, price_thb: int, category: str) -> str:
      prompt = f"""เขียนรีวิว {product_name} ({category}) ราคา {price_thb}฿ สำหรับ fur parent ไทย
      format: hook 1 ประโยค, pros 3 ข้อ, cons 2 ข้อ, แนะนำให้ใครซื้อ, CTA ลิงก์ Shopee
      ภาษาไทย informal 200 คำ ห้ามใช้ภาษาทางการเกินไป"""
      
      msg = client.messages.create(
          model="claude-sonnet-5",
          max_tokens=400,
          messages=[{"role": "user", "content": prompt}]
      )
      return msg.content[0].text

  products = [
      ("Royal Canin Indoor Adult 2kg", 650, "อาหารแมว"),
      ("Catit Senses 2.0 Digger", 450, "ของเล่นแมว"),
      ("Tractive GPS Cat Tracker", 2200, "GPS tracker"),
  ]

  for name, price, cat in products:
      print(f"\n--- {name} ---")
      print(generate_pet_review(name, price, cat))
  ```
- **ทำไมขายได้:** content 3 โพสต์ใน 5 นาที vs 6 ชั่วโมงทำมือ — ถ้า affiliate commission แค่ 5% ต่อ 500฿ = 25฿/sale แต่ถ้า scale 100 posts/เดือน + 2% CTR = คุ้มค่า time investment มาก
